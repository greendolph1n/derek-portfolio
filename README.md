Personal portfolio website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.


## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel (deployment)

## Features

- Modern responsive design
- Project showcase
- Contact links

## Development

Install dependencies:

```bash
npm install
```
Run locally:

```bash
npm run dev
```
Deployment

The site is deployed with Vercel and automatically redeploys on every push to main.
